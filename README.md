# blog

> blog for dadatang

## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm start
```

## 开发流程
1. 项目初始化（done）

3. 页面布局开发
4. 安装mysql，后端接口开发
